UseDwmFlushIfApplicable(OpenGLContextHandle, bool)

Sets whether calls to SwapBuffers(OpenGLContextHandle) should use DwmFlush() to sync if DWM compositing is enabled. This can improve vsync performance on systems with multiple monitors using different refresh rates, but is likely to break in a multi-window scenario. If using multiple windows, only one window should have this property set.

By default this value is set to false.
